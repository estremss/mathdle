About the project

Mathdle is an interactive math testing platform that supports students during assessments by integrating a built-in AI chat assistant. When a student faces a challenging question and doesn't know the answer, instead of searching through books or external resources, they can simply ask the embedded AI chat for a hint or explanation. This smart assistant provides helpful guidance without giving away direct answers, encouraging learning while offering real-time support inside the test environment.

Our goals

  • Enhance Learning Efficiency
  • Reduce Test Anxiety
  • Improve Student Confidence
  • Encourage Consistent Practice
  • Support for Students and Teachers

Implemented technologies

  • HTML&CSS
  • Python
  • Flask
  • Mistral API
  • Virtual environment (venv)
